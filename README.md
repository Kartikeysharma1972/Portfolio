# Kartikey Sharma — Portfolio

Personal developer portfolio website showcasing AI/ML projects, skills, and professional experience.

## Live

[iamjustk.site](https://iamjustk.site)

## Tech Stack

- HTML5, CSS3, JavaScript
- Responsive design with custom animations
- Deployed with custom domain

## Setup

```bash
git clone https://github.com/Kartikeysharma1972/Portfolio.git
cd Portfolio
# Open index.html in browser, or serve with any static server
npx serve .
```
